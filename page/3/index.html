算法导论笔记(7)：散列表和散列函数

发表于 2017-11-16 | 分类于 算法导论学习笔记 | 阅读次数:

散列表(Hash table, 哈希表)是一种根据键(Key)直接访问内存存储位置的数据结构，也即它通过计算一个关于键值的函数，将所需查询的数据映射到表中一个位置来访问记录，这加快了查找速度。 这个映射函数称做散列函数，存放记录的数组称做散列表。散列表在实际应用中的性能是极好的，尽管在最坏的情况下查找一个元素的时间为\(\Theta\left( n\right)\)，在合理的假设下，散列表平均查找时间为\(\Theta\left( 1 \right)\)! 本文将介绍散列表的基本概念和两种最简单的散列函数——除法散列函数和乘法散列函数。

算法导论笔记(6):动态规划之最长公共子序列问题

发表于 2017-11-14 | 分类于 算法导论学习笔记 | 阅读次数:

动态规划如之前介绍的分治策略一样，是一种算法设计方法，它常应用于子问题重叠的情况，即不同的子问题具有公共的子子问题。动态规划算法对每个子问题只求解一次，将其解保存在一个表格中，从而无需梅西求解一个个子子问题时都重新计算。动态规划方法通常用于求解最优化问题（optimization problem）,这类问题可以有很多个解，每个解都有一个值，我们希望寻找其中的一个最优解（注意，最优解可能不唯一）。本文通过最长公共子序列（Longest Comment Subsequence, LCS）问题为例来对动态规划方法深入的认识。
